### namestack 추가 설정 정의
* Outbox 테이블은 무한히 커지나요?
  * A: Namastack Outbox는 처리 완료된 레코드를 자동으로 정리합니다. 필요 시 보관 기간을 설정할 수 있습니다. -> 설정 방법

  * 관련 설정
~~~yaml
outbox:
  processing:
    delete-completed-records: true   # 완료된 레코드 자동 삭제 (기본값: false)
~~~

   * 설정 옵션
     * false : (기본값) 처리 완료된 레코드를 테이블에 유지
     * true : 처리 완료 후 레코드를 자동으로 삭제

* TraceContextProvider
  * java 25의 scope value 적용 여부 검토
*  테스트 코드 상세 설명 문서 작성
* micrometer, telemetry
